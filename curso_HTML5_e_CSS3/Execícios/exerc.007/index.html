Principais Formatações

Negrito ou Destaque

Nesta frase temos um termo em negrito usando a tag <b> (não semântica).

Nesta frase temos um termo em destaque usando a tag <strong>(semântica).

Itálico ou Ênfase

Nesta frase temos um termo em itálico usando a tag <i> (não semântica).

Nesta frase temos um termo em ênfase usando a tag <em> (semântica).

Texto Marcado

Podemos também criar um texto marcado usando a tag <mark>.

Eno outro parágrafo temos outro texto marcado no final.

Texto grande e pequeno

Estamos criando um texto grande com a tag <big> (não semântica) e um texto pequeno com a tag <small> (semântica) neste parágrafo.

Texto deletado

Podemos marcar um texto como excluído usando a tag <del>, para indicar que ele deve ser lido, mas não considerado.

Texto inserido

Podemos marcar um texto como inseridousando a tag <ins>, para dar uma ênfase e indicar que ele foi adicionado posteriomente.

Existe também a possibilidade de criar um textosublinhado usando a tag <u> (não semântica).
